Restaurant Landing Page

This project is a responsive restaurant landing page built using HTML, CSS, JavaScript, and Bootstrap.

Features
- Responsive design for mobile, tablet, and desktop
- Sticky navigation bar
- Hero section
- Menu highlights
- Image gallery
- Customer testimonials
- Contact form with validation
- Google map integration
- Dark mode toggle
- Today's special feature using JavaScript

How to Run

1. Download the project folder
2. Open index.html in any browser

Technologies Used
HTML
CSS
JavaScript
Bootstrap
